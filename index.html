Como iniciar o jogo?

  1. Para iniciar o jogo é obrigatório colocar um nome no jogador 1 e 2.
  2. Para selecionar os personagens, você deverá clicar em jogador 1 e em seguida escolher um personagem. Repita o mesmo processo para escolher o jogador 2.
  3. Por fim, aperte no botão "Entre no jogo".

Como Jogar:

  1. O jogador da esquerda é o primeiro a jogar, os simbolos usados nas marcações das jogadas irão variar de acordo com o personagem escolhido.
  2. Pelo andar do jogo um jogador irá responder uma questão aleatória de história sobre "Revolução Francesa".
  3. Se o jogador acertar a pergunta ele ganha um acerto e se errar, ele perde a vez da jogada.
  4. Quando um jogador ganhar a partida ele terá que responder uma pergunta sobre o personagem do adversário. Se ele acertar, o adversário perde uma vida. Se ele errar, apenas ele perde uma vida.
  5. Perde o jogo, o jogador que zerar suas vidas primeiro.
